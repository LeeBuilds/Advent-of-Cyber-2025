**Linux**

*cat welcome.txt* to read the text file.
*ls -la* to show all files. The -a flag shows the hidden files. The -l flag shows the additional details, such as file permissions and file owner.
*echo 'Hello World'* print text to the terminal.
/var/log/ linux directory containg all logs
*grep* a command to look for a specific text inside a file. usage *grep "Failed password" auth.log*, *grep "hello" file.txt*
*find* a command that searches for files with specific parameters, such as *-name*. usage *find /path -name "filename.txt"* more info: https://man7.org/linux/man-pages/man1/find.1.html
Files with the .sh extension contain CLI commands and are called shell scripts. Such scripts are used both by IT teams to automate things and by attackers to quickly run malicious commands.
*cat wishlist.txt | sort | uniq* lists unique items from the wishlist.txt.
*mv eastmas.txt wishlist.txt* replaces the original file with eastmas.txt.


**cmd**

*dir* to list visible files.
*dir /a* to reveal hidden ones.
*type filename.txt* Display the contents of a text file
*type file1.txt file2.txt > combined.txt* combine files
*type file1.txt file2.txt* to view multiple files
